AKAI S5000 Sys Ex control notes
——————————————————————

Control Message format

0xf0	start of sysex
0x47	Akai ID
0x5e	S5000/6000 model
0x..	Device ID (bits 0..5) | User Ref config (bits 6..7)
0x..	User Ref (followed by extra user refs depending on config)

Confirmation Message format

0xf0	start of sysex
0x47	Akai ID
0x5e	S5000/6000 model
0x..	Device ID (bits 0..5) | User Ref config (bits 6..7)
0x..	User Ref (followed by extra user refs depending on config)
0x..	Reply ID (O = OK, D = Done, R = Reply, E = Error)
0x..	Section
0x..	Item
0x..	Data MSB (note 7-bit bytes)
0x..	Data LSB (note 7-bit bytes)
0x..	Checksum (if enabled)
0xf7	end of sysex

Sections

0	sysex config
2	system setup
4	midi config
6	keygroup zone
8	keygroup
10	program
12	multi
14	sample tools
16	disk tools
18	multi fx
20	scene list
22	midi song file
32	front panel control
42	alt program
44	alt multi
46	alt sample tools
50	alt multi fx
56	alt keygroup zone (block)
58	alt keygroup (block)
60	alt program (block)
62	alt multi (block)
68+	reserved
